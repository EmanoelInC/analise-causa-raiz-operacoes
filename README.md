# Análise de Causa Raiz e Otimização de Processos (Estudo de Caso)

Repositório focado na aplicação prática de metodologias ágeis de qualidade (**PDCA**, **Diagrama de Ishikawa** e **Princípio de Pareto**) utilizando análise estatística descritiva com Python.

## 🎯 Aplicação Prática
Análise focada em mitigar as perdas decorrentes de retornos de rua e devoluções comerciais. O script realiza a classificação automática dos motivos críticos de recusa, permitindo que a liderança comercial tome decisões preditivas sobre rotas e renegociação de débitos na ponta.
